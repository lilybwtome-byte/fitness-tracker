# 💪 健身记录小程序

一个纯 HTML 的健身记录 + 饮食追踪应用，手机/电脑都能用。

[👉 立即使用](https://lilybwtome-byte.github.io/fitness-tracker/)

## 功能

- 📅 日历视图 — 月历格子，颜色标记训练类型（背/胸/臀腿/游泳/休息）
- 🏋️ 训练记录 — 三分化训练，预设动作模板，自动暂存草稿
- 🍽️ 饮食追踪 — per-meal macro 目标，食物搜索+自定义食物
- 📊 数据统计 — 训练次数/趋势/体重变化图表

## 使用方式

1. 用手机浏览器打开上面的链接
2. iOS Safari 可「添加到主屏幕」，体验接近原生 App
3. 所有数据存在手机浏览器 localStorage 中，不会上传到任何服务器
